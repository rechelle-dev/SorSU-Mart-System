# Firebase Studio

This is a NextJS starter in Firebase Studio.

To get started, take a look at src/app/page.tsx.

AS A ADMIN
Create Seller Accounts:
   From the Admin Dashboard, go to the Users section.
   Click "Add User".
   Fill out the form, making sure to select the "Seller" role and provide a store name. The seller can then log in with the credentials you provided.
Approve Products:
   Go to the Products section in the Admin Dashboard.
   Here you will see a list of all products submitted by sellers. You can Approve or Reject them to make them visible (or not) in the main store.
    
AS A SELLER
Log In: Log in using a seller account created by the admin.
Manage Your Store: You will be taken to the Seller Dashboard. From here you can:
   Add New Products: Go to the Products tab and click "Add New Product". These products will go to the admin for approval.
   View Orders: See incoming orders from shoppers in the Orders tab.
   Check Analytics: View your sales performance in the Analytics tab.
   
AS A SHOPPER
Browse and Buy: Any visitor can sign up for a "Shopper" account. They can:
   Browse all approved products on the homepage and category pages.
   Add items to their cart and wishlist.
   Complete the purchase using the checkout page.
   View their past orders in their profile.
